# superstore-sales-performance-dashboard
End-to-End Business Intelligence Project using SQL, Excel, and Power BI to analyze Superstore sales performance.

## Superstore Sales Performance Analysis Dashboard

### Project Overview

This project is part of the Data Analytics & Business Intelligence Internship – Task 3. The objective is to analyze the Superstore sales dataset and develop a complete Business Intelligence workflow including SQL analysis, Excel variance analysis, and an interactive Power BI dashboard.

The project demonstrates how raw sales data can be transformed into meaningful business insights using modern analytics tools.

### Project Objectives

1. Analyze sales performance across regions, product categories, and customer segments
2. Identify profitability drivers and underperforming areas
3. Calculate variance and growth metrics
4. Develop an interactive Power BI executive dashboard
5. Provide data-driven business recommendations

### Tools & Technologies Used

- Tool	Purpose

1. PostgreSQL	Data querying and analysis
2. Excel	Variance and growth analysis
3. Power BI	Interactive dashboard visualization
4. GitHub	Project documentation and version control

The project uses the Superstore Sales Dataset, which contains information about:

### Key Business Insights

1. The business generated 14.92M in total sales with a 10.2% profit margin.
2. The South region shows the highest profit margin, indicating strong operational efficiency.
3. Technology products generate the highest profit contribution.
4. Corporate customers contribute the largest share of revenue.
5. High discount levels significantly reduce average profit margins.

### Strategic Recommendations

1. Based on the analysis, the following strategies are recommended:
2. Improve profitability in the West region
3. Focus marketing efforts on high-profit product categories
4. Optimize discount policies
5. Strengthen relationships with high-value corporate customers
6. Investigate low-margin products such as Furniture
